## 🚀 三步上线

### 第一步：替换微信二维码

1. 打开 `index.html`，找到 `id="qrcodeArea"` 的区域
2. 把你的**微信个人二维码截图**保存为 `qrcode.png`
3. 放在同一文件夹，替换 HTML 中该区域为：
   ```html
   <img src="qrcode.png" style="width:160px;border-radius:8px;">
   ```

> 💡 微信二维码获取方法：微信 → 我 → 点击顶部二维码图标 → 截图

---

### 第二步：部署上线（二选一）

#### 方案A：CloudStudio 一键部署（推荐，有公网链接）

让 AI 助手帮你执行部署命令，部署后获得一个**公网链接**，打印成二维码贴在档口。

#### 方案B：本地直接用（零成本）

1. 双击打开 `index.html`（用浏览器打开）
2. 把地址栏的链接发给朋友测试
3. 如需公网访问，使用 **CloudStudio 部署**

---

### 第三步：生成二维码贴档口

1. 部署成功后会得到一个链接，比如：`https://xxx.cloudstudio.io`
2. 用任意**二维码生成器**（微信小程序搜"二维码生成"）把链接转成二维码
3. 打印出来，贴在档口显眼位置
4. 立牌文案建议：
   > **📱 扫码预约下单**  
   > 免排队 · 保新鲜 · 到店即取

## 📋 订单怎么收到？

顾客下单流程：
1. 扫码打开小程序
2. 选商品 → 填预约信息 → 提交
3. 成功页点 **「复制订单信息发微信」**
4. 打开微信，粘贴发给**你的个人微信**

你收到后：
- 确认订单
- 按预约时间备货
- 顾客到店报订单号即可取货

## 🔧 进阶：让订单自动通知你（可选）

目前方案是顾客手动发微信，适合起步阶段。

如果想让订单**自动发到你微信**，有两种免费方案：

### 方案1：接入金数据（推荐）

1. 注册 [jinshuju.net](https://jinshuju.net)（免费版）
2. 创建一个"订单收集"表单
3. 把 `app.js` 里的 `submitOrder` 函数改成向金数据提交
4. 金数据支持**微信通知**、**邮件通知**、**短信通知**

### 方案2：接入 EmailJS（免费200封/月）

1. 注册 [emailjs.com](https://emailjs.com)
2. 绑定你的邮箱或微信（通过邮件转发）
3. 在 `app.js` 里调用 EmailJS SDK 发送订单邮件

## 💡 使用小贴士

1. **每天营业前**：打开 `admin.html`，清空昨日已完成订单
2. **高峰期**：把手机放在旁边，及时确认订单
3. **顾客不会用**：耐心教一遍，大多数人都用过美团/饿了么，操作一样
4. **预约时段**：根据实际产能设置，避免接太多做不过来
5. **复购顾客**：加微信，下次直接微信下单（更方便）

## 🆘 常见问题

**Q：顾客说打不开链接？**  
A：检查是否已部署到公网，本地文件不能直接分享。

**Q：订单信息没收到？**  
A：目前依赖顾客手动发微信，记得提醒顾客点"复制订单信息发微信"。

**Q：怎么修改价格和菜品？**  
A：直接编辑 `index.html`，找到对应产品卡片修改文字和价格。

**Q：能支持在线支付吗？**  
A：当前版本是到店付款。如需在线支付，可接入微信支付（需要企业资质）或使用**收款码**（顾客下单后展示你的收款码）。
